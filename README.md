# BooksByMax Landing Page

Author landing page for **BooksByMax Designs** - A static HTML website showcasing children's activity books with direct links to Amazon.

## 🎯 Purpose
Single-page author hub that funnels traffic to Amazon product pages for BooksByMax activity books.

## 🎨 Design
- **Layout**: Category-focused with featured section
- **Colors**: Rainbow Bright (#4299e1 blue, #f6e05e yellow, #ed8936 orange)
- **Style**: Kid-friendly, playful, engaging

## 📁 Structure
```
BooksByMax-Landing/
├── index.html              # Main website (final design)
├── README.md              # This file
├── books_master_30.csv    # Complete book database (10/30 filled)
├── DESIGN_CONCEPTS.md     # Design exploration documentation
├── MANUAL_DATA_GATHERING_GUIDE.md # Instructions for adding books
├── assets/
│   └── covers/            # Book cover images
└── docs/                  # Project documentation
```

## 🚀 Features
- **Featured Books Section**: Highlights 3 top sellers
- **Category Organization**: Books grouped by type (Coloring, Journals, Cookbooks)
- **Mobile Responsive**: Works on all devices
- **Direct Amazon Links**: Each book links to its Amazon product page
- **Sales Stats Display**: Shows units sold and ratings

## 📝 How to Add More Books
1. Open `books_master_30.csv` in spreadsheet software
2. Fill rows 11-30 with book data:
   - ASIN (Amazon product ID)
   - Title
   - Category
   - Age Range
   - Description
   - Amazon URL
3. Download cover images to `assets/covers/`
4. Update `index.html` with new book entries
